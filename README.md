# F1 Interactive Website

This repository contains a fully functional **Formula 1 themed website** featuring 3D car models, team statistics, and championship standings.  
The project uses **Three.js** to render interactive F1 cars and provide an immersive visual experience.

The site includes **4 main pages**, each designed to present different aspects of the Formula 1 season.

---

## Features Overview

### **Home Page**
The homepage introduces the website and includes:
- a hero section with the site's overview  
- an embedded **video of the latest Grand Prix**  
- the **current state of the championship** (drivers & teams highlights)  

---

### **Teams Page (3D Experience)**
This page uses **Three.js** to render F1 car models for each team.

Includes:
- interactive 3D models of team cars  
- information about each team  
- list of drivers  
- points earned during the season  
- team history and basic background info  

Users can rotate and inspect the cars in real time.

---

### **Drivers Championship Page**
A dedicated page showing the **full drivers' standings**:
- driver names  
- their teams  
- total points  
- ranking  
- photos or driver graphics (if provided)  

This page focuses on the individual competition throughout the season.

---

### **Constructors Championship Page**
This page displays the **constructors (teams) standings**, including:
- all F1 teams  
- their total points  
- ranking  
- season performance summary
